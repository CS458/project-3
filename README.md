# project-3

To run backend server:
- Install python 3.10
- `cd backend`
- `pip install requirements.txt`
- `python main.py`

To run frontend server:
- Install `nodeJs` LTS version.
- `cd frontend`
- `npm i`
- `npm run dev`

To test:
- Run backend server.
- After running backend server, run frontend server.
- `python test.py`
